## 1. Рекурсия

**Задача 1.1.** 
Важность: :star::star::star::star::star:
Сложность: :fire:
***

> Напишите функцию ```sumTo(n)```, которая для данного ```n``` вычисляет сумму чисел от 1 до ```n```, например:

```
sumTo(1) = 1
sumTo(2) = 2 + 1 = 3
sumTo(3) = 3 + 2 + 1 = 6
sumTo(4) = 4 + 3 + 2 + 1 = 10
...
sumTo(100) = 100 + 99 + ... + 2 + 1 = 5050
```

Источник: https://learn.javascript.ru/recursion
